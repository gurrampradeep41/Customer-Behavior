# Customer-Behavior
A data analytics and visualization project designed to analyze customer purchasing behavior, subscription trends, sales performance, and revenue insights using interactive dashboards. This project helps businesses understand customer preferences, identify profitable product categories, and make data-driven decision
# Features
Customer overview with total customers, average purchase amount, and average review rating
Subscription status analysis (Subscribed vs Non-Subscribed customers)
Revenue analysis by product category
Sales analysis by category
Customer segmentation based on age groups
Shipping type filtering
Gender-based analysis
Interactive and visually appealing dashboard design
# Dashboard Insights
Clothing category generates the highest sales and revenue
Majority of customers are non-subscribers
Young and middle-aged customers contribute most to sales
Customer behavior varies across categories and shipping preferences
